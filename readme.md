# Tiny Serial Monitor 2

Tiny Serial Monitor - a simple cross platform serial terminal

## Installation

Install with:

```bash
sudo npm install -g toptensoftware/tsm2
```

## Usage

```bash
tsm2 /dev/ttyACM0
```

## More information

Use `tsm2 --help` for more.
